## Usage

### Docker

```shell-session
$ docker run --rm -p 1234:80 ntrrg/site
```

Go to <http://localhost:1234/> with a browser.

### Hugo

**Requirements:**

* Hugo >= 0.52

```shell-session
$ hugo server
```

Go to <http://localhost:1313/> with a browser.

## Edition

**Requirements:**

* GNU Make
* Docker >= 18.03

```shell-session
$ make run
```

```shell-session
$ make lint
```

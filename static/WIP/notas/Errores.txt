Linux:
	* Error de variables locales al intentar instalar algun paquete:
		# dpkg-reconfigure locales

	* Error actualización "python2.7-minimal"
		1. Descargar el paquete:
			$ apt-get download python2.7-minimal

		2. Reinstalar el paquete descargado:
			# dpkg -i --force-overwrite python2.7-minimal* && rm python2.7-minimal*

	* Resolución de pantalla:
		1. Instalar el paquete de utilidades de tarjetas de video
			# apt install mesa-utils

		2. Crear la nueva resolución
			# cvt <ancho> <alto>

			Nota: Copiar la línea que empieza con la palabra "Modeline" de la salida del comando.

		3. Crear/Modificar el archivo de configuración gráfica
			# <editor> /etc/X11/xorg.conf

			Section "Device"
				Identifier "Configured Video Device"
			EndSection

			Section "Monitor"
				Identifier "Configured Monitor"
				Vendorname "Generic LCD Display"
				Modelname "LCD Panel <ancho>x<alto>"
				Horizsync 31.5-64.0
				Vertrefresh 56.0 - 65.0
				<lineaModeline>
			Gamma 1.0
				# modeline "640x480@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
				# modeline "800x600@56" 36.0 800 824 896 1024 600 601 603 625 +hsync +vsync
				# modeline "800x600@60" 40.0 800 840 968 1056 600 601 605 628 +hsync +vsync
				# modeline "1024x768@60" 65.0 1024 1048 1184 1344 768 771 777 806 -vsync -hsync
				# <lineaModeline>
			EndSection

			Section "Screen"
				Identifier "Default Screen"
				Monitor "Configured Monitor"
				Device "Configured Video Device"
				DefaultDepth 24
				SubSection "Display"
					Depth 1
					Modes "<ancho>x<alto>"
				EndSubSection
				SubSection "Display"
					Depth 4
					Modes "<ancho>x<alto>"
				EndSubSection
				SubSection "Display"
					Depth 8
					Modes "<ancho>x<alto>"
				EndSubSection
				SubSection "Display"
					Depth 15
					Modes "<ancho>x<alto>"
				EndSubSection
				SubSection "Display"
					Depth 16
					Modes "<ancho>x<alto>"
				EndSubSection
				SubSection "Display"
					Depth 24
					Modes "<ancho>x<alto>"
				EndSubSection
			EndSection

	* Tarjeta de red inactiva:
		1. Agregar la interfaz eth0:
			# echo -ne "\nauto eth0\niface eth0 inet dhcp" >> /etc/network/interfaces

		2. Reiniciar el servicio de red:
			# service networking restart

		3. Hacer que la interfaz eth0 busque una dirección IP:
			# dhclient eth0

	* Error de segmentación con uGet:
		# rm -r `find / -name uGet`

Windows:
	* Hora desincronizada entre sistemas operativos:
		1. Abrir el editor de registros.
		2. Ir a la rura "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\TimeZoneInformation".
		3. Agregar un valor DWORD (32 bits) con nombre "RealTimeIsUniversal".
		4. Asignar un valor hexadecimal de 1 al elemento que se creó.

	* Cursor se congela al pulsar alguna tecla:
		1. Desinstalar aplicaciones relacionadas al uso del ratón.

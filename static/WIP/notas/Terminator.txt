Instalar:
	# apt-get install terminator

Atajos del teclado (Hotkeys):
	· Copiar: Ctrl + Shift + C
	· Pegar: Ctrl + Shift + V
	. Finalizar proceso activo: Ctrl + C
	. Borrar todo despues del cursor: Ctrl + K
	. Borrar todo antes del cursor: Ctrl + U
	. Buscar entre el historial de comandos: Ctrl + R
	. Historial de comandos:
		Anterior: Arriba
		Siguiente: Abajo

	Pestañas y terminales:
		· Nueva ventana: Ctrl + Shift + I
		· Cerrar ventana: Ctrl + Shift + Q
		· Nueva pestaña: Ctrl + Shift + T
		. Nuevo terminal horizontal: Ctrl + Shift + O
		. Nuevo terminal vertical: Ctrl + Shift + E
		· Cerrar terminal: Ctrl + Shift + W
		. Expandir/Reducir terminal: Ctrl + Shift + X
		. Pantalla completa: F11

		. Zoom:
			Aumentar: Ctrl + +
			Predeterminado: Ctrl + 0
			Reducir: Ctrl + -

		· Cambiar terminal activo:
			Anterior: Ctrl + Shift + Tab
			Siguiente: Ctrl + Tab
			Abajo: Alt + Abajo
			Arriba: Alt + Arriba
			Derecha: Alt + Derecha
			Izquierda: Alt + Izquierda

		. Redimencionar terminal activo:
			Abajo: Ctrl + Shift + Abajo
			Arriba: Ctrl + Shift + Arriba
			Derecha: Ctrl + Shift + Derecha
			Izquierda: Ctrl + Shift + Izquierda

		. Activar multiples terminales:
			Pestaña actual:
				Activar: Super/Windows + T
				Desactivar: Super/Windows + Shift + T

			Todos:
				Activar: Super/Windows + G
				Desactivar: Super/Windows + Shift + G

		. Cambiar pestaña activa:
			Anterior: Ctrl + RePág
			Siguiente: Ctrl + AvPág

		· Mover pestañas:
			Derecha: Ctrl + Shift + AvPág
			Izquierda: Ctrl + Shift + RePág

**ntWeb** es un pequeño sitio web con grandes intenciones.

## Uso

**Requerimientos:**

* Hugo >= 0.55

```shell-session
$ hugo server
```

Ir a <http://localhost:1313/> con un navegador web.

### Docker

```shell-session
$ docker run --rm -p 1313:80 ntrrg/ntweb
```

Ir a <http://localhost:1313/> con un navegador web.
